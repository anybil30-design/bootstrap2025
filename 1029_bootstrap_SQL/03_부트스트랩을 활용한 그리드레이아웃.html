03_부트스트랩을 활용한 그리드레이아웃

부트스트랩은 하나의 행에 반드시 12개의 열(컬럼)을 사용해야 합니다.

만약 12개의 열(컬럼)을 모두 사용하지 않으려면, 더 넓은 열(컬럼)을 만들기 위해 열(컬럼)들을 하나의 그룹으로 만들어 사용할 수 있습니다.

그리드 시스템에서는 화면의 너비별 5개의 클래스를 가지고 있습니다. 이미지 참고

각 해상도별 클래스명 이해하기

  • 해상도 576미만일 경우 클래스명 없음
  • 해상도 576이상일 경우 클래스명 'sm' - 모바일 버전
  • 해상도 768이상일 경우 클래스명 'md' - 태블릿 작은 버전
  • 해상도 992이상일 경우 클래스명 'lg' - 태블릿 큰 버전
  • 해상도 1200이상일 경우 클래스명 'xl' - PC버전 일반버전
  • 해상도 1400이상일 경우 클래스명 'xxl' - PC버전 큰사이즈
